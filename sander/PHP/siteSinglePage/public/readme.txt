.htaccess - перенаправляет все запросы на "index.php" используя модуль "mod_rewrite" (для web-сервера Apache).
index.php - скрипт - отправная точка приложения. В простых сайтах всю логику можно положить в этот скрипт. 
            В сложных приложениях требуется разбиние приложения на куски, которые ложатся в "../app/*".
            Можно использовать свои скрипты, но лучше воспользоваться готовым фреймворком.
